# Employee Management System - Java

Aplicación desarrollada en Java aplicando Programación Orientada a Objetos.

## Descripción

Sistema de gestión de empleados que permite:

- Crear empleados
- Modificar datos
- Calcular sueldo
- Gestionar empleados a tiempo completo y por horas

## Tecnologías

- Java
- Programación Orientada a Objetos
- Herencia
- Polimorfismo
- Clases abstractas
- ArrayList

## Conceptos aplicados

- `Main.java` - Punto de entrada
- `GestionEmpleado.java` - Lógica principal del sistema
- Clase abstracta `Empleado`
- Subclases:
  - `EmpleadoTiempoCompleto`
  - `EmpleadoPorHoras`
- Sobreescritura de métodos
- Encapsulamiento
- Gestión mediante referencias del tipo padre

Proyecto académico desarrollado durante el ciclo DAM.
